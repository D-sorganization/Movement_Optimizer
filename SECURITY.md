# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities to the repository maintainers via a private GitHub issue or email.

## Security Scanning

This repository uses:
- `bandit` for static security analysis
- `pip-audit` for dependency vulnerability scanning
